## Before You Start

<Checklist>
  <li>Rust stable is installed through `rustup`</li>
  <li>Node.js 22 or newer is installed</li>
  <li>`pnpm` 10 or newer is available</li>
  <li>Docker and Docker Compose are running</li>
</Checklist>

## Fast Path

```bash
pnpm install
pnpm infra:up
cp .env.example .env
pnpm dev
curl http://127.0.0.1:3001/health
```

Expected response:

```json
{"status":"ok","ready":true,"postgres":{"status":"ok","detail":null},"redis":{"status":"ok","detail":null}}
```

Local URLs:

- API: `http://localhost:3001`
- Web: `http://localhost:3000`
- Docs: `http://127.0.0.1:3002`
- Landing: `http://localhost:3003`

## Seeded Demo Path

If you want a fuller tenant without modeling everything by hand:

```bash
pnpm dev:seed
```

Default seeded users:

- `admin@qryvanta.local` / `admin`
- `user@qryvanta.local` / `admin`
