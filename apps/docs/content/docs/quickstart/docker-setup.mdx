---
title: Docker Setup
description: Start local infrastructure required for API and web development.
---

## Prerequisites

- Docker Desktop (or Docker Engine) with Compose support
- Port `5432` available

## Start Infrastructure

```bash
docker compose up -d
```

This starts:

- PostgreSQL 16 on `127.0.0.1:5432`

## Bootstrap Local Env

```bash
cp .env.example .env
```

Before using auth flows, set these values in `.env`:

- `AUTH_BOOTSTRAP_TOKEN`
- `SESSION_SECRET`
- `WEBAUTHN_RP_ID`
- `WEBAUTHN_RP_ORIGIN`

## Verify Health

```bash
docker compose ps
```

`qryvanta-postgres` should show healthy status before running API/web apps.

## Stop Infrastructure

```bash
docker compose down
```
