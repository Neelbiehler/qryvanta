## Operator Baseline

<Checklist>
  <li>Health checks are monitored for API, database, Redis, and worker runtime where used</li>
  <li>Secrets are not stored in source control and rotation rules are documented</li>
  <li>Backups are scheduled and restore paths are tested</li>
  <li>Trusted proxy and browser-facing header behavior are validated in staging</li>
  <li>Queued workflow mode has at least one healthy worker and the right coordination settings</li>
</Checklist>
