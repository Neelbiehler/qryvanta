This page summarizes the active platform roadmap from `docs/ROADMAP.md`.

## Enterprise Foundation Track

- Status: Complete
- Tenant-scoped RBAC checks in application use-cases
- Append-only audit logs for state-changing operations
- Default owner-role assignment during tenant bootstrap

## Phase 1

- Status: Complete
- Persistent metadata storage and migrations
- Auth bootstrap and tenancy primitives
- Replace in-memory repository with durable adapter

## Phase 2

- Status: Complete
- Rich metadata model and publish lifecycle
- Runtime record APIs from published metadata
- Validation engine for dynamic schemas

## Phase 3

- Status: In progress
- App builder UX for entities, views, and forms
- Metadata-driven list and form rendering

## Phase 4

- Status: In progress
- Workflow triggers, actions, retries, and run history
- Queued worker topology with authenticated internal claim channel

## Phase 5

- Status: Backlog
- RBAC, tenant isolation hardening, and audit trails

## Phase 6

- Status: Backlog
- Self-hosting packaging, backup/restore, and operations docs

## Current focus

- Active story: ST-07 Optional remote workflow workers with secure execution boundaries
- Active epic: EP-03 Phase 4 Automation Engine and Worker Topology
